Algorithms

About Prim's Algorithm

A greedy algorithm that grows a single MST by adding the cheapest edge connecting a node in the tree to a node outside the tree.

Analogy: Powering one house, then finding the shortest wire to connect a new house to your powered network.

  • Time Complexity: O(E log V)
  • Best for: Dense graphs.

About Kruskal's Algorithm

A greedy algorithm that builds an MST by adding the lowest-weight edges that do not form a cycle.

Analogy: Building the cheapest bridges first, but skipping any that connect two islands already reachable from each other.

  • Time Complexity: O(E log E)
  • Best for: Sparse graphs.
